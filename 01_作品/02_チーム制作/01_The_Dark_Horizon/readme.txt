◆動作確認済み環境◆

（DirectxInput対応のゲームパッド(ジョイスティック)に対応しています。）

◆操作方法◆
< キー操作 >
[ W ]		前移動
[ A ]		左移動
[ S ]		後移動
[ D ]		右移動

[ I ]		視点を左に移動
[ J ]		視点を上に移動
[ K ]		視点を下に移動
[ L ]		視点を右に移動

[ V ]		射撃

[ F ]		インタラクト

[ SPACE ]	ジャンプ

< パッド操作 >
[ Lスティック ]		移動
[ Rスティック ]		視点移動

[ A ]				ジャンプ
[ X ]				インタラクト

[ UP ]				上選択
[ DOWN ]			下選択

[ RT ]				射撃

[ START ]			ポーズ

< マウス操作 >
[ BUTTON L ]		射撃

[ カーソル移動 ]	視点操作
