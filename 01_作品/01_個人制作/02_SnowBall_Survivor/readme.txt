◆作品について◆

◇使用言語 : C
◇SDK      : DirectX9
◇開発期間 : 2024/12～2025/01(1か月)
◇ジャンル : 2Dアクション

◆動作確認済み環境◆
Windows11
（DirectxInput対応のゲームパッド(ジョイスティック)に対応しています。）

◆操作方法◆
< キー操作 >
[ W ]		奥移動
[ A ]		左移動
[ S ]		手前移動
[ D ]		右移動
[ SPACE ]	ジャンプ
[ ENTER ]	玉を作る
[ V ]		雪玉を投げる
[ B ]		氷玉を投げる
[ N ]		砂利玉を投げる

< ゲームパッド操作 >
[ Lスティック ]	移動
[ A ]			ジャンプ
[ R1 ]			玉を作る
[ X ]			雪玉を投げる
[ Y ]			氷玉を投げる
[ B ]			砂利玉を投げる
