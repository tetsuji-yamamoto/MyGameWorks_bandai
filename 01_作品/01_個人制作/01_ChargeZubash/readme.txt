◆作品について◆

◇使用言語 : C
◇SDK      : DirectX9
◇開発期間 : 2024/10～2042/11(1か月)
◇ジャンル : 2Dアクション

◆動作確認済み環境◆
Windows11
（DirectxInput対応のゲームパッド(ジョイスティック)に対応しています。）

◆操作方法◆
< キー操作 >
[ A ]		左移動
[ D ]		右移動
[ SPACE ]	ジャンプ
[ E ]		攻撃
[ F ]		エネルギー放出攻撃

< ゲームパッド操作 >
[ Lスティック ]	移動
[ A ]			ジャンプ
[ X ]			攻撃
[ Y ]			エネルギー放出攻撃
